# 驾校预约管理系统 java291

## 项目概述

驾校预约管理系统是一个基于Spring Boot开发的平台，旨在为驾校管理人员、教练以及学员提供便捷的在线服务。系统主要包括后台管理功能和前台学员功能两部分，涵盖用户管理、内容管理、预约管理和考试管理等多个业务模块。

## 技术栈

- 后端：Spring Boot
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

### 后台管理

#### 驾校管理员功能

- **用户管理**
  - 管理员管理：查看、新增、修改、删除管理员
  - 教练管理：查看、新增、修改、删除教练信息
  - 学员管理：查看、新增、修改、删除学员信息
- **内容管理**
  - 驾校公告管理：发布和管理驾校公告
  - 留言板管理：查看和管理用户留言
  - 学习资料管理：管理学习资料、留言和收藏
  - 轮播图管理：管理首页轮播图
- **预约管理**
  - 教练预约管理：查看和管理学员预约信息
- **考试管理**
  - 试卷管理：创建和管理考试试卷
  - 试题管理：管理考试题目
  - 考试记录：查看学员考试记录
  - 错题本：查看学员错题记录
- **数据管理**
  - 管理公告类型、科目、资料类型、组卷方式等基础数据

#### 驾校教练功能

- **教学管理**
  - 驾校公告：查看驾校公告
  - 教练预约：查看和审核学员预约
  - 留言板：查看和回复学员留言
  - 学习资料：查看教学资料
- **考试管理**
  - 试卷管理：查看考试试卷
  - 试题管理：查看考试题目

### 前台（面向驾校学员）

- **个人信息管理**
  - 查看和修改个人资料
- **教练预约**
  - 查看可预约的教练信息
  - 进行教练预约
- **学习资源管理**
  - 查看驾考宝典（学习资料）
  - 管理个人收藏的学习资料
- **考试相关**
  - 查看考试信息
  - 查看个人考试记录
  - 查看错题本
- **互动交流**
  - 查看驾校公告
  - 使用留言板进行交流

## 系统角色

- 驾校管理员
- 驾校教练
- 驾校学员

## 运行环境

- 操作系统：Windows/Linux
- 数据库：MySQL 5.7+
- JDK版本：1.8+

- 提供全面的角色权限管理，保证系统安全
- 集成便捷的预约管理，提高教练与学员的沟通效率
- 完善的考试管理功能，助力学员学习进度跟踪
- 清晰的学习资源管理，帮助学员高效复习
- 互动交流平台，促进驾校内部信息流通
